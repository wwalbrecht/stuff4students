# stuff4students

Sammlung von Jupyter Notebooks und Ressourcen für Studierende zu verschiedenen Themen.

## Inhaltsübersicht

- **jupyter-intro.ipynb**: Einführung in Jupyter Notebooks und deren Features wie Magics und Diagramme.
- **hello_colab_cpp.ipynb**: Einstiegs-Notebook für C++ Programmierung in Google Colab.
- **gitIntro.ipynb**: Einführungs-Tutorial zu Git mit praktischen Beispielen, Branches und Merge-Konzepten.
- **cpp2py.ipynb**: Überblick über die Grundlagen von Python mit Vergleichen zu C++ Konzepten.
- **C++09-Klassen-Teil2.ipynb**: (in Arbeit) Tutorial zu objektorientierten Konzepten und Klassen in C++.
- **swt-tests.ipynb**: Lehrmaterial zu Softwaretesting mit Fokus auf Unit-Tests, Test-Frameworks und Code Coverage in Python.
